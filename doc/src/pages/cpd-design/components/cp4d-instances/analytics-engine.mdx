# Analytics engine powered by Apache Spark Instances

Analytics Engine instances can be defined by adding the `instances` section to the `cartridges` entry of catridige `analytics-engine`. The following example shows the configuration to define an instance.

```
cp4d:
- project: cpd-instance
  openshift_cluster_name: "{{ env_id }}"
...
  cartridges:
  - name: analytics-engine
    size: small
    state: installed
    instances:
    - name: analyticsengine-instance
      storage_size_gb: 50
```


| Property | Description | Mandatory | Allowed Values |
| -------- | ----------- | ---------- | ------------- |
| name | Name of the instance | Yes | |
| storage_size_db | Size of the storage allocated to the instance | Yes | numeric value |
